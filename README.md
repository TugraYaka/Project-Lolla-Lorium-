# Lolla Engine & Lorium Agent Hub

Lolla is a modern, cross-platform hub for managing and running window-based AI agents and applications.

## 🚀 Getting Started

To run Lolla Engine locally, follow these steps:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your system.

### 2. Installation
Clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/TugraYaka/Project-Lolla-Lorium-.git
cd Project-Lolla-Lorium-
npm install
```

### 3. Running the App
Start the engine:

```bash
npm start
```

## 📦 Packages

Lolla uses a dynamic package system. Packages are located in the `packages/` directory. Each package must have a `manifest.json` file.

### Adding Lorium Agent
To install the Lorium Agent package:
```bash
npm install @tugrayaka/lorium-agent
```
